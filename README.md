## About the project 

This website is tailored towards programmers in attempt to train your typing speed with programming phrases rather then passage prompts offered by traditional typing test websites, with a custom "programming" like feel for the UI.

This project was created by Mohamed Al-Nassirat for the HackED Beta 2022 Hackathon.

## Live Demo

A live version of this website is avaliable for demo hosted directly on Github Pages and can be found here: https://mohamedal-nassirat.github.io/ProgoType/


## Roadmap

Creating this project originally in only 24 hours as a solo was a challenge and I was not able to complete all the features I wished to implement, as such I wish to add the following:
  - Integrate actual competitive programming phrases/promtps rather then random sample code
  - Add menu selection for different langugages (currently only support Python, C, GO, Javascript on refresh)
  - Add daily leaderboard system
